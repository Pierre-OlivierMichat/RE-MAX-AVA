# Assets Folder

This folder contains the images and SVG files used in the Hero Section component.

## Required Assets

The following assets need to be placed in this folder:

### Images
- `client-1.png` - Client profile image 1
- `client-2.png` - Client profile image 2
- `client-3.png` - Client profile image 3
- `building.png` - Main building image for hero section

### SVG Icons
- `star-mask-1.svg` - Star icon mask layer 1
- `star-mask-2.svg` - Star icon mask layer 2
- `star-content.svg` - Star icon content
- `plus-icon.svg` - Plus icon for client avatars
- `arrow-down.svg` - Down arrow icon
- `arrow-diagonal.svg` - Diagonal arrow icon for button
- `shape-icon.svg` - Decorative shape icon

## Getting Assets from Figma

These assets are available from the Figma Desktop plugin. The original URLs were:
- Images served from: `http://localhost:3845/assets/`

You can download these assets directly from Figma or replace them with your own assets.
